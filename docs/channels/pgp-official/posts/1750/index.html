18 марта 2026 «Пепеляев Групп» совместно с Уполномоченным по защите прав предпринимателей в Приморском крае, Корпорацией развития Дальнего Востока и Арктики проведут вебинар и расскажут о налоговых преференциях для инвесторов.

⏱️Время: 18 марта с 15:00 до 16:30 (время по Владивостоку)

🔸Программа вебинара:

➡️Ключевые изменения для резидентов специальных инвестиционных режимов: новые гарантии для международных ТОР и участников СЗПК, расширение полномочий регионов по инвестиционным налоговым вычетам и новые ограничения для федеральных инвестиционных налоговых вычетов.
➡️Механизм ограничения льгот с 01.04.2026: формула расчёта ограничений, последствия несоблюдения, исключения, переходные правила для действующих резидентов.
➡️Совместимость специальных инвестиционных режимов: позиция Минфина.
➡️Риски для застройщиков – резидентов СПВ: кто вправе проверять правомерность присвоения статуса резидента.
➡️Судебная практика по ТОР, СПВ и ОЭЗ: практические примеры оснований для доначислений, расторжения соглашений и взыскания льгот.

🔸Спикеры:

Сергей Скалий, заместитель генерального директора по сопровождению инвестиционных проектов Корпорации развития Дальнего Востока и Арктики,
Егор Лысенко, руководитель Сибирского отделения «Пепеляев Групп» (г. Красноярск),
Сергей Сосновский, партнер налоговой практики «Пепеляев Групп» (г. Санкт-Петербург),
Евгения Демьянчук, юрист Дальневосточного отделения «Пепеляев Групп» (г. Владивосток).

🔸Модератор:

Наталия Кошелева, врио Уполномоченного по защите прав предпринимателей в Приморском крае.
