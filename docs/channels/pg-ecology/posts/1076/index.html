🌞 Уважаемые участники канала, доброго дня!

Мы подготовили обзор судебной практики для EcoStandard.Journal по спорам о возмещении вреда окружающей среде, в связи с размещением отходов грунта. В обзор включили позиции судов:

🔹о правовых основаниях для взыскания денежной компенсации вреда;
🔹о взыскании вреда за размещение загрязненного грунта;
🔹о том, в каких случаях размещение грунта на стройплощадке не является основанием для привлечения к имущественной ответственности.

Также в обзоре приведены комментарии по правовым позициям судов, для их практического применения.
