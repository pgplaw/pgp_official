📣 Разработан Проект Постановления Правительства РФ о внесении изменений в постановление Правительства РФ от 29.12.2018 г. N 1730, которым утверждены таксы и методика для определения размера денежной компенсации вреда лесам.

Согласно Пояснительной записке, Проект предусматривает установление единых ставок платы для всех регионов страны в целях выработки консолидированного подхода при расчете ущерба от незаконной рубки насаждений, произрастающих на землях неразграниченной собственности.
