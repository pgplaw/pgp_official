🌐 SAMR представило проект антимонопольных рекомендаций для интернет-платформ
 
Проект рекомендаций по антимонопольному комплаенсу для интернет-платформ открыт для общественных обсуждений до 29 ноября. Об этом сообщается на сайте Антимонопольного центра БРИКС.
 
Государственное управление по регулированию рынка Китая (SAMR) представило проект руководства по антимонопольному комплаенсу для интернет-платформ. Документ призван помочь операторам выявлять риски, связанные с монопольными соглашениями, и рекомендует избегать заключения таких соглашений, организации их заключения третьими сторонами или «предоставления материальной помощи» при работе платформ или собственных сервисов.
 
❗️Для снижения рисков злоупотребления доминирующим положением проект правил советует платформам со значительной рыночной властью регулярно оценивать наличие доминирования и воздерживаться от действий, которые могут свидетельствовать о злоупотреблении доминирующим положением.
 
В документе также представлены восемь типичных риск-сценариев, включая «несправедливое ценообразование», продажи «ниже себестоимости», блокировку аккаунтов, требования «выбора одного из двух» и «гарантию самой низкой цены по всей сети».
