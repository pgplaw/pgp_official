🌐 SAMR опубликовала руководство по комплаенс-менеджменту для интернет-платформ

➡️ Размещены Руководящие мнения по действиям, способствующим эффективному управлению комплаенсом в компаниях-операторах интернет-платформ. Об этом в обзоре № 8 новостей китайского антитраста сообщил Антимонопольный центр БРИКС.

В Мнениях подчеркивается необходимость:

🟠укреплять структурную организацию корпоративных систем комплаенс-менеджмента;
🟠повышать ответственность руководителей компаний за эффективность комплаенс-менеджмента;
🟠стимулировать компании к найму квалифицированных специалистов по комплаенсу;
🟠формировать перечни должностных обязанностей специалистов по комплаенсу;
🟠помогать компаниям в разработке списков комплаенс-рисков;
🟠оказывать компаниям поддержку в проведении тренингов и тестирований по комплаенсу, а также разработке систем поощрений и наказаний для сотрудников системы комплаенс-менеджмента.
