🌐 SAMR выпускает рекомендации по комиссиям для онлайн-платформ

➡️ Новые рекомендации направлены на снижение финансовой нагрузки на продавцов и повышение прозрачности расчета комиссий. Об этом сообщается на сайте Антимонопольного центра БРИКС.

🟠Государственное управление по регулированию рынка Китая (SAMR) в субботу представило новый свод руководящих принципов по соблюдению правил взимания комиссий и сборов на онлайн-платформах. Цель нововведений — дальнейшее регулирование способов, с помощью которых платформы электронной коммерции взимают с операторов комиссии, сервисные сборы, плату за членство и другие расходы, включая техническую поддержку, маркетинг и услуги, связанные с данными.
