​🔸Право на оспаривание приватизации ограничат десятилетним периодом

Президентский совет по кодификации поддержал законопроект Минэкономики о введении «срока давности» для приватизации. Иски об истребовании имущества нельзя будет подавать спустя 10 лет с момента его выбытия из госсобственности. 10-летний период будет применяться не только для будущих требований, но и для уже существующих, судебные решения по которым еще не вступили в силу. В РСПП назвали это решение ключевым для инвестклимата.

Благодаря снижению этих затрат мы существенно уменьшили финансовую нагрузку на предприятие, что позволило трудоустроить на сегодня 135 работников».
