​🔸 Власти ограничат льготы для резидентов особых режимов

В Госдуму внесен законопроект, который может серьезно изменить правила игры для инвесторов. Власти предлагают ограничить налоговые льготы для резидентов ТОР, ОЭЗ и других режимов, привязав их к объему реальных инвестиций и выполнению обязательств по соглашениям.

Эта мера, по мнению авторов, направлена на борьбу с низкой инвестиционной активностью и «сверхдоходностью» у отдельных компаний.

🔸 Часть параметров ФИНВ закрепят в Налоговом кодексе

Минфин России готовит поправки в НК РФ, чтобы перенести часть параметров федерального инвестиционного налогового вычета (ФИНВ) из постановления Правительства в сам кодекс. Это должно обеспечить большую стабильность и предсказуемость для бизнеса.

🔸 Запущен калькулятор для оценки эффективности капвложений

Минэкономразвития России запустило новый онлайн-инструмент - калькулятор для оценки эффективности капитальных вложений. Он поможет инвесторам заранее проверить свои проекты на соответствие требованиям и избежать формальных ошибок при подаче заявок на господдержку.
