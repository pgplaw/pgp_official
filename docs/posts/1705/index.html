⚡️ Регистрация в апартаментах

Владельцы апартаментов не могли в них зарегистрироваться по месту пребывания – правила предусматривали регистрацию только для жилья или гостиниц.

🔘 КС РФ принял Постановление, которым разрешена временная регистрация граждан в апартаментах, сходных с квартирами и не входящих в номерной фонд гостиниц или иных средств размещения.


❗️ Важно

Важно учитывать Постановление КС РФ при проектировании апартаментов, поскольку акцент сделан на характеристиках, «сходных с квартирой» (изолированность, планировка, мини-кухня, санузел), которые будут проверять для регистрации.

Необходимо сформировать для каждого объекта и передавать покупателям пакет документов, подтверждающих пригодность здания и конкретного помещения для пребывания граждан (проектная декларация, особенности планировки), – они понадобятся для регистрации.
